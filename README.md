# workoutCounter

Apple Watch용 운동 세트/횟수 카운터 앱입니다.

## 앱 소개

헬스장에서 운동할 때 세트와 횟수를 손쉽게 기록할 수 있는 Apple Watch 앱입니다. 가속도계를 이용해 동작을 자동으로 감지하거나, 버튼으로 직접 카운트할 수 있습니다.

## 주요 기능

- **자동 횟수 카운팅** — 가속도계로 운동 동작을 감지해 횟수를 자동으로 카운트
- **수동 카운팅** — `-` / `+` 버튼으로 직접 횟수 조절 가능
- **휴식 타이머** — 세트 완료 후 설정한 시간만큼 휴식 타이머 자동 시작
- **운동 기록** — 세트별 횟수를 저장하고 이전 기록 확인 가능
- **햅틱 피드백** — 횟수 증가 및 휴식 타이머 종료 시 진동 알림

## 지원 운동

- 벤치 프레스
- 랫 풀다운
- 시티드 로우
- 오버헤드 프레스
- 레터럴 레이즈
- 프론트 레이즈
- 바이셉 컬
- 스쿼트

## 사용 방법

1. 앱을 실행하면 운동 목록이 표시됩니다
2. 원하는 운동을 선택합니다
3. 운동을 시작하면 가속도계가 동작을 자동 감지해 횟수를 카운트합니다
4. 세트가 끝나면 `→` 버튼을 눌러 다음 세트로 넘어갑니다
5. 휴식 타이머가 자동으로 시작되며, 종료 시 진동으로 알려줍니다
6. 운동을 마치려면 `✕` 버튼을 눌러 기록을 저장합니다

## 설정

- 휴식 시간은 설정 화면에서 조절할 수 있습니다
